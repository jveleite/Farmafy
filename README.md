# 💊 Farmafy - Sistema de Farmácia

Sistema web de farmácia desenvolvido com foco em portfólio, simulando funcionalidades reais de um sistema comercial.

---

## 🚀 Tecnologias utilizadas

- React (Vite)
- Supabase (PostgreSQL)
- JavaScript

---

## 📦 Funcionalidades atuais

- Cadastro de produtos
- Listagem de produtos em tempo real
- Integração com banco de dados

---
